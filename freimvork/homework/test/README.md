## Приложение - простая тестовая система.

1. Спроектирован интерфейс, включающий отображение одного вопроса и вариантов ответа

2. При выборе ответа и нажатии кнопки "Ответить" проверка правильности ответа и смена вопроса на экране

3. Вопросы теста читается из файла. Имя файла закрепить в программе (выбор файла добавим позже). Формат файла: Вопрос начинается с префикса q:, варианты ответов с номера (1.,2., 3., ...), правильный ответ помечен знаком "+" в конце текста ответа. (Можете придумать свою вариацию формата файла)

4. В тесте 8 вопросов.

5. Когда получены ответы на все вопросы,  на интерфейсе показаны количество правильных ответов, процент правильных ответов и оценка
